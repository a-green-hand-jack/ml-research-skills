# Paragraph Openings and Closings

Use this file when a section needs concrete paragraph starts, endings, or local flow. Treat the examples as templates, not fixed language.

## Problem Opening

Use at the start of an introduction or section.

Patterns:

- "A central difficulty in [setting] is that [specific failure], which makes [desired capability] hard to evaluate or achieve."
- "Current [methods/evaluations/theories] work well when [condition], but break down when [claim-relevant condition]."
- "The key question is whether [hypothesis] holds under [setting]."

Avoid:

- "Recently, deep learning has achieved great success..."
- "With the rapid development of..."

## Method Opening

Use at the start of a method section or component paragraph.

Patterns:

- "We design [component] to address [failure mode] by [mechanism]."
- "The role of [component] is to [function], which is necessary because [claim-relevant reason]."
- "This section describes [method/artifact] in the order needed to understand [testable claim]."

## Result Opening

Use before discussing a figure or table.

Patterns:

- "Table X tests whether [claim] holds in [setting]."
- "Figure X isolates [factor], allowing us to distinguish [our explanation] from [alternative explanation]."
- "We first evaluate [primary claim] on [benchmark], where the key question is whether [specific property]."
- "The main comparison is against [baseline family], which represents [alternative or prior capability]."

Avoid:

- "We report results in Table X."
- "As shown in Figure X, our method performs well."

## Paragraph Closing

Use to make a paragraph hand off to the next idea.

Patterns:

- "This motivates [next method component / experiment / analysis]."
- "This result supports [claim], but leaves open whether [next question]."
- "We therefore test [mechanism/boundary] directly in [next experiment]."
- "The next section turns this design principle into [method/protocol/theorem]."

Avoid:

- closing with a generic restatement
- introducing an unsupported new claim in the final sentence
