# Decision Framing

Advisor updates should make decisions easy.

## Decision Block

```markdown
## Decision Needed: [short question]

Recommendation: [option]

Options:
- A: [description]
- B: [description]
- C: [description]

Evidence:
- [fact/result/link]

Risks:
- If we choose A:
- If we choose B:
- If we wait:

Ask:
- Please decide whether we should [specific action].

Next action after decision:
- If yes:
- If no:
```

## Good Asks

- "Should we target NeurIPS with the current empirical story, or spend two more weeks on baseline coverage?"
- "Can we narrow the claim to mechanism analysis instead of SOTA performance?"
- "Is this negative result enough to stop this direction?"
- "Which baseline is mandatory before I write the next draft?"

## Weak Asks

- "Any thoughts?"
- "What should I do?"
- "Is this good?"

Rewrite weak asks into options plus a recommendation.
