# Advisor Update Template

```markdown
# Advisor Update - YYYY-MM-DD

## One-Line Status

## What Changed Since Last Update

## Evidence

| Item | Evidence | Interpretation | Confidence |
|---|---|---|---|

## Blockers and Risks

| Risk/blocker | Impact | Proposed response | Owner |
|---|---|---|---|

## Decision or Feedback Needed

## Plan Until Next Update

## Links
```

## Email Version

```markdown
Subject: [Project] update + decision on [topic]

Hi [Name],

One-line status: ...

The main update is ...

The decision I need is ...

My recommendation is ...

Next, I will ...
```

Keep the email version short. Link to reports instead of embedding long tables.
