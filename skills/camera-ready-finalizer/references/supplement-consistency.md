# Supplement and Appendix Consistency

Use this when the paper has an appendix, supplement, checklist, or extra material.

## Cross-Reference Checks

- main paper references to appendix sections resolve
- appendix figure/table/equation labels compile
- numbering is stable after final edits
- supplement file names match upload requirements
- captions and table titles match main paper terminology

## Content Checks

- extra experiments use the same method names and metrics as the main paper
- appendix results do not contradict main claims without explanation
- proof assumptions match method and limitation text
- ablation details match the reported main-table numbers
- hyperparameters and reproducibility details match code or configs
- limitations and broader-impact text reflect final paper scope

## Reviewer Requested Details

If reviewers requested details, make them findable:

- add a clear appendix section heading
- mention the appendix location in the main paper if appropriate
- avoid burying promised results in unlabeled tables
