# De-Anonymization

Use this after acceptance to restore final paper identity and metadata.

## Restore

- author names and order
- affiliations and emails
- equal contribution or advising footnotes
- acknowledgements
- funding, compute, grant, and institutional support
- code, data, model, benchmark, project page, and artifact links
- self-citations and previous-work wording
- author contributions if required

## Remove

- "anonymous authors"
- "submitted under review"
- anonymized repository placeholders
- hidden TODOs about anonymity
- masked institution names
- anonymized acknowledgements
- anonymized self-citation workarounds
- draft-only comments or macros

## Metadata

Check when relevant:

- PDF title and author metadata
- final camera-ready title/abstract fields in upload system
- supplementary file names
- code repository title and citation metadata
- project page author/institution display

Do not invent personal, institutional, funding, or author-order details. Ask the user for missing required metadata.
